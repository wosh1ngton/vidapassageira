# vidapassageira
Aplicação para registro e planejamento de viagens

Utiliza keycloak para autenticação e DeepSeek como ferramenta de IA para gerar sugestões para as viagens.

A aplicação faz sugestões do tipo: "Onde Ficar", "Como chegar", "Onde Ir", "Onde Comer", 
tudo isso utilizando IA. Permite o cadastro estruturado de um itinerário de viagem partindo da sugestão da IA.
